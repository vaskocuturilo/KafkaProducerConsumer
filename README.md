##   

The demo project for Kafka working with Golang and Java.

### This project was created to demonstrate how to work with the Kafka.

- Create a REST API services (send/pull message to/from Kafka).
- Add Apache Kafka.
- Add Kafka UI.
- Add Docker Compose.
- Add unit(repository, service and rest) and integration tests with testcontainers.

You will need the following technologies available to try it out:

* Git
* Spring Boot 3+
* Gradle 9+
* JDK 24+
* Golang 1.25+
* Apache Kafka
* Kafbat UI (for Apache Kafka)
* Docker compose
* IDE of your choice


### How to run via Spring Boot.

``` ./gradlew bootRun ```


### How to run via Golang.

``` go run .```

### How to run via Docker Compose.

``` docker-compose up --detach```
